# 📊 Sales Metrics Reporting (SMR) Dashboard

A professional business analytics dashboard built using Streamlit that integrates Excel-based data pipelines and provides real-time visualization of key performance metrics.

---

## 🚀 Features

* 📂 Multi-sheet Excel integration (Data, Staging, Summarized, Dashboard)
* 📊 Real-time KPI tracking (Sales, Profit, Investment, Capital)
* 📈 Interactive data visualization using Streamlit
* 🔗 Embedded Excel dashboard view (exact layout using OneDrive/Google Sheets)
* 🧩 Modular ETL pipeline design (Data → Staging → Summary → Dashboard)

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Excel (ETL Pipeline)

---

SMR_Dashboard/

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/smr-dashboard.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
